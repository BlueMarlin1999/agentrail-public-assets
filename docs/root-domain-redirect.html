Recommended

Route agentrail.tech cleanly to the public www host.

Your live branded site already runs on https://www.agentrail.tech/ . The cleanest finish is to redirect the root domain agentrail.tech to the www host or keep the GitHub Pages A-record setup until apex DNS fully stabilizes.

Spaceship click-by-click path

  1. Open the Advanced DNS page for agentrail.tech.
  2. Switch to URL Redirect.
  3. Add a redirect record.
  4. Set Host to @.
  5. Set target URL to https://www.agentrail.tech .
  6. Choose 301 Permanent.
  7. Save and apply the record.

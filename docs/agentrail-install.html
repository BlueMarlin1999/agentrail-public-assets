一行安装

默认安装到 default profile：

curl -fsSL https://www.agentrail.tech/install-online.sh | bash

安装到指定 profile：

curl -fsSL https://www.agentrail.tech/install-online.sh | bash -s -- myprofile

怎么工作

STEP 01 AgentRail 接收工作项，按角色分工把任务送入底层执行链路。
STEP 02 Perpetual Engine 驱动状态机推进，负责实现、检查、测试、评审、回退重试和阻塞治理。
STEP 03 Operator 可以查看摘要、恢复执行、重置重试计数，并对失败阶段做人工介入。

把 agentrail.tech 跳转到 www.agentrail.tech

你现在已经把正式站点跑在 www.agentrail.tech。最省事、最稳定、最像正式产品的做法，就是让裸域名 agentrail.tech 做 301 跳转到 https://www.agentrail.tech。

Spaceship 按键级步骤

  1. 打开 agentrail.tech 的“高级 DNS”页面。
  2. 切到“URL 重定向”。
  3. 点击“添加重定向”。
  4. Host 填：@
  5. 目标 URL 填：https://www.agentrail.tech
  6. 重定向类型选：301 Permanent
  7. 保存 / 应用。

如果你的面板不支持根域名 URL redirect，就继续保留 4 条 GitHub Pages A 记录，等根域名直接生效。

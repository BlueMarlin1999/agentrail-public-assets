Install AgentRail, download the full release bundle, and verify the runtime in one place.

One-line install

Install into the default profile:

curl -fsSL https://www.agentrail.tech/install-online.sh | bash

Install into a named profile:

curl -fsSL https://www.agentrail.tech/install-online.sh | bash -s -- myprofile

Problem it solves

Agent work is easy to start and hard to trust. AgentRail gives each task a visible state, a recovery path, and enough evidence that I can decide whether a website update is ready to merge.

Where it fits

Dispatch

Use Antigravity or a similar surface to shape the request, then let AgentRail hold the workitem, scope, acceptance criteria, and next action.

Workers

Send implementation or review work to Codex, Claude Code, or OpenCode while keeping the state machine and evidence outside the worker's chat window.

Operator control

When a task fails or needs judgment, the blocked state, retry count, changed files, and checks remain visible enough to resume or stop cleanly.

What this unlocks in practice

Parallel delivery

Route separate workstreams to different coding agents without losing stage control, audit trails, or operator visibility.

Best-tool routing

Use Claude Code where reasoning depth matters, Codex where repository automation matters, and OpenCode where provider flexibility matters.

Safer operations

When any worker fails, AgentRail preserves failure boundaries, blocked states, retry counts, and resume paths instead of pretending everything succeeded.
